## Heavy Sniper Rifle (w/ modded variants)

### Info:
A DKS-501 sniper rifle with a thicker barrel and a differently shaped stock.
I adjusted some pixels and made a few edits to the stock and lens colors.

Made by **NotDefix**, edited by **NovaRain**

### Tools used:
* Titanium FRM Browser 1.3
* Frame Animator 2.70.1
* [Paint.NET](https://www.getpaint.net)
* [GIMP](https://www.gimp.org)
* [PNGOUT](http://advsys.net/ken/utils.htm)
